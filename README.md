# FAMBF — Калькулятор продукта с защитой капитала (USDT) + участие в росте BTC

Брендированный калькулятор структурного продукта: **100% (или частичная) защита тела в USDT + участие в росте BTC**. Один самодостаточный файл `index.html` — без сборки, без зависимостей, без бэкенда. Открывается в любом браузере и работает на GitHub Pages.

## Что считает

- **Бюджет на опцион** и **максимально возможное участие** при заданной защите тела (модель «надёжная нога + колл», principal-protected note).
- **Маржа проекта** — фиксируется на старте и не зависит от направления BTC (хедж на купленных опционах).
- **Сценарии** падения/роста BTC (−50% … +100%): сколько получает клиент и сколько зарабатывает проект.
- **График выплаты** клиента против «BTC в лоб».

## Параметры, которые можно менять

Взнос, цена BTC, срок, подразумеваемая волатильность, надёжная доходность USDT, **уровень защиты тела** (главный рычаг участия), участие, комиссии на входе и с прибыли, буфер на проскальзывание хеджа, кэп на рост (call-spread).

**Живые котировки.** Переключатель «Живая цена опциона с биржи» позволяет подставить реальную цену ATM-колла (и колла на страйке кэпа) с Deribit/OTC вместо модельной цены Блэка–Шоулза — для точного расчёта перед запуском.

## Готовые пакеты (пресеты в один клик)

| Пакет | Срок | Защита тела | Участие (ориентир) | Мин. вход |
|---|---|---|---|---|
| Conservative | 30 дн | 98,5% | ≈ 18% | 10 000 USDT |
| Balanced | 90 дн | 97% | ≈ 30% | 25 000 USDT |
| Growth | 180 дн | 95% | ≈ 40% | 50 000 USDT |

Проценты участия рассчитаны для волатильности 50% и доходности 5%; при других допущениях модель пересчитает их и подсветит, если участие выходит за бюджет.

## Как опубликовать на GitHub Pages

1. Создайте репозиторий и загрузите в него `index.html` (и при желании этот `README.md`).
2. **Settings → Pages → Build and deployment → Source: Deploy from a branch.**
3. Выберите ветку `main` и папку `/ (root)`, нажмите **Save**.
4. Через минуту страница будет доступна по адресу `https://<ваш-логин>.github.io/<репозиторий>/`.
5. Эту ссылку можно передавать заказчикам — всё считается у них в браузере, ничего никуда не отправляется.

Альтернативно файл можно просто открыть локально (двойной клик) или разместить на любом статическом хостинге (Netlify, Cloudflare Pages, Vercel).

## Важно

Инструмент для оценки структуры продукта — **не котировка и не инвестиционная рекомендация**. Проценты участия зависят от рыночной волатильности и ставок на дату запуска и **не являются обещанием доходности**. Защита тела действует только на дату погашения и обеспечивается надёжной ногой и купленными опционами; её надёжность ограничена платёжеспособностью контрагентов и биржи, а также устойчивостью USDT. Любое предложение с обещанием возврата капитала может быть регулируемым продуктом — перед запуском требуется юридическая проверка лицензирования в соответствующей юрисдикции.
